# Image Credits

## Placeholder Images (Phase 01)

All images in `public/assets/images/` are **placeholder / development-only** assets sourced during Phase 01 and should be swapped for proper licensed content before any commercial publication.

### Source: LoremFlickr

13 of the 14 images were downloaded from [loremflickr.com](https://loremflickr.com) — a service that wraps keyword-based CC-licensed Flickr photos. Default license from LoremFlickr is **CC BY 2.0** (attribution required).

> **Note — deviation from original plan:** Validation decision #3 specified Unsplash CC0. However, Unsplash Source API (`source.unsplash.com`) was officially deprecated in mid-2024 and now returns `HTTP 503`. LoremFlickr was chosen as the closest free+legal+keyword-searchable substitute. Attribution is looser than CC0 but the workflow remains free-of-charge and production-unsafe; final project deployment should replace these with properly licensed assets.
